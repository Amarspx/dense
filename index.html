Dense is a jQuery plugin that offers an easy way to serve device pixel ratio aware images. Clear the blurry, images be crisp.

This responsive image is implemented by simply including the plugin on the page and calling its $.fn.dense() initializer method.

<script src="jquery.min.js"></script>
<script src="dense.min.js"></script>
<script>
    $('img').dense();
</script>

Options

$('selector').dense([String method] [, Object options])

Dense offers few methods and options that can be used to both customize the plugin's functionality and return resulting values. All interaction is done through the $.fn.dense() method, that accepts a called method and its options object as its arguments. Both arguments are optional, and either one can be omitted. The following methods are available.

init

$('selector').dense([Object options]) -> this

Renders a pixel-ratio-aware image. The init method is the default that is called if no method argument is specified.

Options

ping
A prefix added to the generated links. Default is true. Boolean
dimension
Whether to keep the original dimensions or remove. Default is false. Boolean

Example

$('img.retina').dense({
    'ping'      : false,
    'dimension' : true
});

updateDimensions

$('selector').dense('updateDimensions') -> this

Sets an image's width and height attributes to its native values.

devicePixelRatio

$('selector').dense('devicePixelRatio') -> Integer

Gets device pixel ratio rounded up to the closest integer.

Example

var ratio = $(window).devicePixelRatio();
alert(ratio);
